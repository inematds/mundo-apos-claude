# 🎼 O Mundo Após o Claude

Curso gratuito **INEMA.CLUB** — de usuário a maestro: pare de só conversar com a IA e comece a comandá-la para construir qualquer coisa, até montar o seu próprio Jarvis.

**Acesse:** https://inematds.github.io/mundo-apos-claude/

## Trilhas

1. **🧭 Fundamentos** — a virada mental e o vocabulário, em português de gente.
2. **🛠️ Prática** — mão na massa: dirigir a IA para construir coisas reais.
3. **🤖 Jarvis & Arsenal** — montar seu assistente pessoal + recursos prontos.

## Como funciona

Páginas HTML self-contained (Tailwind via CDN + camada de aprendizagem `assets/learn.*`).
Seu progresso, dúvidas e anotações ficam salvos no próprio navegador. Abre direto, sem build.

Público leigo — não precisa ser programador.
